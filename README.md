# DREAMCAR Rent A Car

A luxury car rental website for Dubai, featuring a premium selection of exotic, sports, and luxury vehicles available for daily, weekly, and monthly rentals — with no deposit required.

## Pages

- **Home** (`index.html`) — Hero section, car search, featured vehicles, rental prices, and brand showcase
- **All Cars** (`all-cars.html`) — Full vehicle catalog with filtering options
- **Daily Offers** (`daily-offers.html`) — Special daily rental deals
- **About Us** (`about.html`) — Company information and values
- **Contact** (`contact.html`) — Contact form and location details

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- Font Awesome icons

## Getting Started

Open `index.html` in a browser — no build step required.
